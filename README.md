# とけてるPOPメーカーβ

販売会で使うPOPを、かんたんに大きく作って表示・印刷できる軽量ツールだよ。

年上の上司でも迷いにくいことと、iPad第3世代のSafariでもできるだけ動くことを重視して作っているよ。

## できること

- タイトル、一言コメント、価格や短い補足を入れてPOPを作れるよ
- `遠くで止める` `商品の横で伝える` `セールで押す` の3つの役割から選べるよ
- ピンク、オレンジ、黄色、水色、黄緑の5色から見た目で選べるよ
- iPad向けの大きい表示モードで、そのまま売り場に置いて使えるよ
- A4に4枚並べた印刷プレビューも使えるよ
- 外部ライブラリなし、サーバーなし、ログインなしで動くよ

## こんな場面向けだよ

- 作業所の販売会
- 季節商品やおすすめ商品の見せ場づくり
- 値札ではなく、商品の魅力やコーナー案内を伝えるPOP

## 使い方

1. `index.html` をブラウザで開くよ
2. 使い方に近い役割を選ぶよ
3. 色を選ぶよ
4. タイトルやコメントを入れるよ
5. プレビューを見ながら、`大きく表示` または `印刷` を使うよ

## 動作方針

- まずは軽さ優先だよ
- 複雑な装飾やアニメーションは入れていないよ
- 1〜2分で使い方がわかる画面を目指しているよ
- ローカルだけで使えるようにしているよ

## ファイル構成

- `index.html` : 画面の土台だよ
- `styles.css` : 見た目と印刷レイアウトだよ
- `app.js` : 入力反映、テンプレ切替、表示更新の処理だよ

## GitHubで公開するときのメモ

このリポジトリは静的ファイルだけで動くよ。

- ローカルで使うなら `index.html` を直接開けばよいよ
- GitHub Pages で公開するなら、`main` ブランチの `/(root)` を公開元にすると使いやすいよ
